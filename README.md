# Steve Belgum Website

This is the personal website for Steve Belgum, built with React. It showcases services for couples and veterans, along with information about his book and contact details.

## Features

- **Services Section**: Information on counseling and support for couples and veterans.
- **Book Promotion**: Link to purchase the upcoming book.
- **Contact Form**: Integration with Google Forms for inquiries.
- **Responsive Design**: Mobile-friendly navigation and layout.

## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd steve-belgum-website
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the Application

To start the development server:

```bash
npm start
```

The application will be available at `http://localhost:3000`.

## Project Structure

- `src/components`: React components (Sections, Contact, BookLink, etc.).
- `src/data`: Content data files (text, links, icons configuration).
- `src/App.css`: Global styles and theming.
